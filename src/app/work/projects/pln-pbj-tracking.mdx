## Overview

PLN PBJ Tracking is a comprehensive procurement management system designed to streamline the workflows between PLN's internal departments and third-party partners. This platform serves as a centralized hub for managing every stage of the procurement process, from initial request to final tracking.

The system provides transparency and efficiency, ensuring that both internal teams and external vendors have a clear view of the procurement status and necessary actions.

## Key Features

- **Procurement Management**: Internal users can easily create, input, and manage new procurement projects within the system.
- **Real-time Status Updates**: Effortlessly update and check the progress of each procurement item, ensuring all parties are informed.
- **Vendor Tracking**: Third-party partners can access the portal to track the status of their specific procurement status in real-time.
- **Detailed Management**: Update and manage specific procurement details to maintain high data accuracy.

## Technologies Used

- **Next.js**: Powering the platform with a robust React framework for high performance and seamless transitions.
- **Shadcn/UI**: Utilizing a modern, accessible component library to create a professional and responsive user interface.
- **Supabase**: Leveraging its powerful backend-as-a-service for secure database management, authentication, and real-time updates.

## Challenges and Learnings

One of the main challenges was designing a permission-based system that provides different interfaces and data access for internal staff versus third-party vendors. Implementing this project helped in mastering the integration of Shadcn components with complex forms and managing real-time data flows with Supabase to ensure that tracking statuses are always up to date.

## Outcome

The platform has significantly improved the efficiency of procurement tracking, reducing the need for manual communication and fragmented documentation. It provides a single source of truth for both PLN and its partners, leading to faster procurement cycles and better transparency.
